# WikiHowQAExtractor
从WikiHow页面抽取中文/英文问答数据

## 用法

```
python wikihow_extract FILE1 FILE2 ... -o wikihow.jsonl
```

## 结果结构

```json
{
    "问题":"...",
    "回答":[
        {
            "回答": "完整回答文本",
            "简要回答": "回答的摘要",
            "结构": {
                ... //回答页面结构，包含方法步骤、小提示、注意事项等
            }
        }
    ]
}
```
